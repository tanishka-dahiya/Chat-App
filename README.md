# Chat-App
## Installation
1)Clone this repository<br/>
for server:
```bash
cd Server
```
```bash
npm i
```
for Front-End:
```bash
cd frontend
```
```bash
npm i
```
## Start
for server:
in server folder
```bash
node index.js
```
for Front-End:
in frontend folder
```bash
npm start
```
## Usage
This Project uses reactjs,ws,node.
